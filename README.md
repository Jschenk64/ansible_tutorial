# ansible_tutorial

This is going to be awesome Ansible tutorial! 

Key Features;
Package Updates:
Updates all installed packages on the system to the latest version.

Apache Installation:
Installs Apache and ensures it is started and enabled.

Firewall Configuration:
Opens port 80 for HTTP traffic permanently and reloads the firewall.

Web Files Deployment:
Downloads the website from GitHub.
Extracts the downloaded ZIP file.
Copies the content to the Apache HTML directory.
Cleans up the temporary files and folders.

Service Management:
Ensures the Apache service is running and enabled to start on boot.
